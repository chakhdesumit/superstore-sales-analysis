# Superstore Sales Analysis

## Objective
To analyze sales data and extract meaningful business insights using Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Visualization

## Analysis Performed
- Monthly Sales Trend
- Regional Sales Distribution
- Category-wise Sales Analysis
- Top Product Analysis

## Key Insights
- Sales increase significantly towards the end of the year, with the highest in November and December.
- February shows the lowest sales, indicating a potential seasonal dip.
- The West region contributes the highest revenue among all regions.
- The South region shows comparatively lower performance.
- A small number of products contribute a large portion of total sales.

## Files Included
- `superstore_sales_analysis.xlsx` → Excel analysis with charts
- `superstore_raw_data.csv` → Raw dataset

## Conclusion
This project demonstrates the use of Excel for data analysis and business insight generation.
